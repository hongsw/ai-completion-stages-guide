# 단계별 체험 체크리스트

## 1단계
- [ ] 챗봇으로 반복 업무 1개를 3회 시도해본다
- [ ] 같은 프롬프트에서 결과 편차를 기록한다

## 2단계
- [ ] n8n 또는 Zapier로 자동화 1개를 만든다
- [ ] 수작업 단계가 몇 개 줄었는지 적는다

## 3단계
- [ ] 파일 읽기/쓰기 가능한 AI 환경을 써본다
- [ ] 웹 검색이나 API 호출이 가능한지 확인한다

## 4단계
- [ ] 조사자 / 작성자 / 검토자 역할을 분리해본다

## 5단계
- [ ] 로그, 상태, 재시도 정책이 있는지 점검한다
- [ ] 실패했을 때 어느 단계에서 깨졌는지 추적 가능한지 본다
