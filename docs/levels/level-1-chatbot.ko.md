# Level 1. 챗봇 사용

## 이 단계의 정의
LLM과 대화하며 생산성을 높이는 가장 기초적인 단계입니다.

## 이 단계에서 얻는 능력
- 요약
- 번역
- 초안 작성
- 아이디어 정리
- 질문을 더 잘하는 법 학습

## 대표 불편
- 매번 사람이 직접 물어봐야 함
- 같은 요청도 결과 편차가 큼
- 내 파일/업무/시스템과 연결되지 않음

## 왜 다음 단계가 필요한가
챗봇은 똑똑하지만, 아직 업무 흐름 안에서 자동으로 움직이지 않습니다.
그래서 자동화가 필요해집니다.

## 관련 clawfit reference level
- **Reference Level 1 — Core comparison targets**
  - 지금 시장에서 많이 쓰이는 대표 도구들을 감 잡는 데 유용합니다.
- **Reference Level 6 — Productivity, input, and human-agent interface tools**
  - 음성 입력, 인터페이스, 실제 사용감 같은 요소를 체감하는 데 연결됩니다.

## 1시간 실습 미션
### 미션
같은 작업을 서로 다른 챗봇 2~3개에 시켜 보고 결과 차이를 비교하세요.

### 추천 작업
- 회의록 요약
- 소개문 작성
- 이메일 초안 작성
- 표 정리

### 실습 절차
1. 같은 프롬프트를 준비합니다.
2. ChatGPT, Claude, Gemini 중 2개 이상에 같은 요청을 넣습니다.
3. 결과를 비교해 아래를 기록합니다.
   - 누가 더 구조적이었는가?
   - 누가 더 자연스러웠는가?
   - 누가 더 쓸모 있었는가?
4. 같은 프롬프트를 한 모델에 3번 반복해 결과 편차도 봅니다.

## 실습 후 질문
- 무엇이 가장 편했나?
- 무엇이 반복적으로 귀찮았나?
- 왜 이 과정을 자동화하고 싶어졌나?
