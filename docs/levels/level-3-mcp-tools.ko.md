# Level 3. 스킬·플러그인·MCP

## 이 단계의 정의
AI가 파일, 검색, API, 데이터베이스 등 외부 도구를 호출하며 실제 행동하는 단계입니다.

## 이 단계에서 얻는 능력
- 파일 읽기/쓰기
- 웹 검색 및 정리
- API 호출
- 외부 서비스와의 실제 상호작용

## 대표 불편
- 아직 한 명짜리 고급 비서에 가깝다
- 긴 프로젝트에서 역할 분리가 약하다
- 복잡한 작업은 협업 구조가 필요하다

## 왜 다음 단계가 필요한가
툴을 잘 쓰는 것과 팀처럼 일하는 것은 다릅니다.
복잡한 문제는 역할을 나누어야 합니다.

## 1시간 실습 미션
### 미션
MCP 또는 툴형 AI 환경이 왜 필요한지 체감하는 데 집중하세요.

### 추천 작업
- 파일 읽기 + 요약 + 저장 흐름 상상/구성
- 검색 + 정리 + 결과물 생성 구조 분석
- MCP 서버 목록을 보며 “내가 당장 쓰고 싶은 도구” 3개 고르기

### 실습 절차
1. MCP Spec 저장소 소개를 빠르게 훑습니다.
2. Awesome MCP Servers 목록에서 흥미로운 서버 3개를 고릅니다.
3. 각 서버가 어떤 실제 불편을 줄여줄지 적습니다.
4. “챗봇과 무엇이 다른가?”를 한 줄로 정리합니다.
